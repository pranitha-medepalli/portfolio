Voice-Activated Personal Assistant

The Voice-Activated Personal Assistant is an AI-powered automation tool built usingPython that enables hands-free interaction with various services. It integrates Google’s Speech Recognition API for voice input, pywhatkit for WhatsApp messaging and YouTube automation, andOpenWeatherMap API for real-time weather updates. The assistant supports multi-language translation using the Google Translate API and can perform tasks like playing music, searchingtheweb, finding nearby places, and retrieving information effortlessly. This project showcases expertisein Python, APIs, automation, and artificial intelligence, making daily tasks more efficient and user friendly.

Book Store Web Application

The Book Store Web Application is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) project designed for seamless online book shopping. It allows users to browse, search, andadd books to their cart, with secure user authentication for login and signup functionalities. The backend, powered by Node.js and Express.js, manages book data and user sessions, while the MongoDB database efficiently stores book details and user information. The frontend, built withReact.js, provides a responsive and user-friendly interface. This project highlights expertise in full- stack development, authentication, database management, and e-commerce functionality.

Airline Reservation System

The Airline Reservation System is a Java-based application designed to facilitate flight booking, ticket management, and reservation tracking. Built using Java AWT, JDBC, and SQL, it allows users to search for available flights, book tickets, view booking details, and cancel reservations. The system ensures efficient data management through an SQL database, handling user and flight records securely. This project demonstrates expertise in Java GUI development, database integration, and real-time reservation handling, making it a practical solution for airline booking management.
